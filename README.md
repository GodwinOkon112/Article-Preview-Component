

# Article Preview Component

## Overview
This project is a simple and interactive Article Preview Component built to simulate a real-world UI feature commonly found in blogs and news websites.

The component displays an article with an image, title, description, and author details. The main functionality centers around a share button, which allows users to reveal or hide social media sharing options dynamically.

### The challenge

The goal of this project is to build a responsive article preview component that allows users to toggle a **share menu** when clicking a button.

Key requirements:

* Display an article with image, title, and description
* Show author profile details
* Implement a share button that reveals social media icons
* Ensure good layout across different screen sizes
* Handle interaction using JavaScript

### What I learned

This project helped me understand how to connect **UI interaction with DOM manipulation**.

Key things I learned:

* How to select elements using `document.querySelector`
* How to attach event listeners to elements
* How to toggle classes dynamically using:

```js
element.classList.toggle("className")
```

* How to structure components into reusable sections (image, text, profile, share UI)

One important takeaway:

> Keeping UI states (like showing/hiding elements) controlled by CSS classes makes your JavaScript cleaner and easier to manage.
